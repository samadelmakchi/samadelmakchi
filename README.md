## 👨‍💻 درباره من

<p align="center">
  <b>صمد المکچی</b> | DevOps Engineer & Backend Developer
</p>

من یک مهندس DevOps و توسعه‌دهنده Backend با بیش از **۲۰ سال تجربه** در اکوسیستم فناوری اطلاعات ایران هستم. تخصص من در نقطه تلاقی **مهندسی نرم‌افزار**، **زیرساخت ابری** و **اتوماسیون** قرار دارد.

> 💡 "به رویکرد Infrastructure as Code و Everything as Code اعتقاد دارم. هدفم خلق سیستم‌های قابل اطمینان، مقیاس‌پذیر و خودکار است."
